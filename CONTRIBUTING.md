# Contributing to @stream-ui

Thanks for your interest! @stream-ui is MIT-licensed and welcomes contributions.

## Setup

```bash
git clone https://github.com/sassy-solutions/stream-ui.git
cd stream-ui
pnpm install
pnpm build
pnpm test
```

## Development

- **Branch from `main`**, name like `feat/...` or `fix/...`.
- Add a [changeset](https://github.com/changesets/changesets) for any user-facing change: `pnpm changeset`.
- Make sure `pnpm lint`, `pnpm typecheck`, `pnpm test`, `pnpm build` all pass.
- Open a PR. CI will run on every push.
