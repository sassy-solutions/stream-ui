# @stream-ui

> Form-first LLM streaming + parsing for **React** and **React Native**. AG-UI + A2UI native. Zero runtime deps. **~16KB total.**

Most LLM chat libraries are chat-first. `@stream-ui` is **form-first**: it makes
embedded, **streaming, validated forms** inside LLM responses a first-class
primitive. Talk to the model, render a form, validate the structured payload,
submit — all as the tokens arrive.

## Packages

| Package | What |
|---|---|
| `@stream-ui/protocol` | AG-UI event types + A2UI v0.9 surfaces + `form` extension. Zero runtime. |
| `@stream-ui/core`     | Transport (SSE/WS), incremental JSON parser, state store, form engine. |
| `@stream-ui/react`    | React hooks + headless `asChild` primitives + RSC helper. |
| `@stream-ui/react-native` | RN hooks + native primitives (`TextInput`, `FlatList`). |
| `@stream-ui/testing`  | Mocks, AG-UI fixtures, RTL helpers. |
